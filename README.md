# 🌟 BabyEnglish - 幼儿英语互动学习

面向 3-6 岁儿童的英语启蒙互动游戏，iPad 优先，支持 PWA 离线使用。

## ✨ 特性

- 🎨 多邻国风格 UI，卡通配色，大触控按钮
- 📱 iPad 优化，竖屏优先
- 🔄 PWA 支持，可离线使用
- 🎮 游戏化学习：关卡地图、星星奖励、连击系统
- 🔤 模块：字母学习、主题词汇、互动练习

## 🛠️ 技术栈

- **框架**: React + TypeScript
- **构建**: Vite 8
- **部署**: Vercel
- **PWA**: vite-plugin-pwa + Workbox
- **样式**: CSS Modules + CSS Variables

## 🚀 开发

```bash
npm install
npm run dev
```

## 📦 构建

```bash
npm run build
npm run preview
```
